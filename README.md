React, Redux, Redux-Saga, TypeScript Project

## Getting Started

Install dependencies:

```bash
$ npm install
```

## Workflow

Start a local development server:

```bash
$ npm start
```

Run unit tests:

```bash
$ npm test
```

Lint and format code:

```bash
$ npm run lint
$ npm run format
```

Build assets for production:

```bash
$ npm run build
```

Run Fake Server

```bash
$ npm install -g json-server
$ npm run mock:api
```
